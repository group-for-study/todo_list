# Turborepo 실행하기

### 1. pakage install

```sh
yarn install
```

기본 패키지 관리는 yarn으로 합니다.

### 2. dev모드 프로젝트 실행

```sh
yarn dev
```

리액트 프로젝트 (web) - 3000포트  
네스트 프로젝트 (api) - 3001포트
